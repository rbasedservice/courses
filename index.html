About the Workshop

Python has been a popular programming language with a constantly growing userbase. Today, Python is being used for each and every aspect of computation; from a simple mathematics project to large scale climate models. Python is intuitive, easy-to-learn and powerful. This introductory course on Python will make you ready to be a part of the Python community; equipping you with the tools and concepts necessary. We make no assumptions about the participants. The course materials have been moulded by keeping a beginner into mind; so anyone with an interest to learn Python can attend this webinar.
We, here at RBS are passionate about disseminating knowledge and we thrive to deliver the best.
